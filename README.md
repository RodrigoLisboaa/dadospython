# Análise da Evolução do IDEB no Brasil (2017, 2019, 2021)

Projeto de análise de dados sobre a evolução do IDEB no Brasil nos anos de **2017, 2019 e 2021**, incluindo recortes por **anos iniciais**, **anos finais** e **ensino médio**, com comparativos por região/UF e visualizações.

## Objetivo
Explorar e comparar a variação do IDEB no Brasil entre 2017–2021, identificando padrões por etapa de ensino e por unidade federativa, gerando gráficos e tabelas de apoio.

## Fonte de dados
Dados públicos do IDEB disponibilizados pelo **INEP**, em planilhas `.xlsx`.

## Tecnologias
- Python
- Pandas
- Matplotlib
- Seaborn
- openpyxl (leitura de `.xlsx`)
